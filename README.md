# BT - Aesthetic Budget Tracker

A beautifully designed, client-side budget tracking web application built entirely with vanilla HTML, CSS, and JavaScript. 

BT is designed to be blazingly fast, fully responsive, and extremely private. It utilizes the browser's `localStorage` to securely save all your financial data directly on your device, meaning there is zero cloud storage and no backend required.

## ✨ Features
- **Local First & Privacy Focused**: All expenses, budgets, and custom tags are stored locally on your device. Your data never leaves your browser.
- **Glassmorphism UI**: A sleek, modern, and highly aesthetic dark mode interface built from scratch without bulky CSS frameworks.
- **Daily & Monthly Tracking**: Easily log day-to-day expenses, attach emojis/tags, and add specific details. Toggle between daily and monthly views.
- **Analytics & Interactive Charts**: Integrated with `Chart.js` to provide colorful, highly distinct pie charts for category breakdowns, and bar charts for month-over-month comparisons.
- **AI-Style Insights**: Get automatically generated insights on your spending, such as percentage increases/decreases compared to the previous month, and your projected daily average.
- **Budgeting**: Set dynamic limits—either an overall monthly budget or specific limits per category—and watch real-time progress bars indicate when you're approaching your limit.

## 🚀 Quick Start
Because this is a completely static site, no build steps or servers are required!
1. Clone the repository.
2. Open `index.html` in your web browser.
3. Start tracking!

*(Perfectly compatible to be hosted directly on GitHub Pages!)*

## 🛠️ Tech Stack
- **HTML5 & CSS3** (Vanilla, CSS Variables, Flexbox/Grid)
- **JavaScript (ES6+)** (Vanilla DOM Manipulation, LocalStorage API)
- **Chart.js** (Loaded via CDN for Analytics)
